# Data Releases

As we source our data from an external (to the project) source, we are dependent on the EES and DfE data cycle. 

Data is normally released in one of three conditions

* Provisional - First run, internal checks and balances made. 
* Revised - Second run, any feedback from establishments fixed and made.
* Final - Last run.
